# 数据分析与数字化实践作品集

吴政达的数据分析与数字化实践作品集。

## 本地运行

```bash
npm install
npm run dev
```

## 当前版本

当前版本面向数据分析相关岗位，突出数据清洗、指标体系设计、BI 可视化、业务分析、数据治理与流程数据化能力。

主项目包括：

- 协定存款业务数据看板
- 审批链智能体

## 替换素材

- 头像：`public/portrait.png`
- 下载简历：`public/resume.pdf`
- 页面内容：`src/data.ts`

公开内容使用脱敏示例数据，不包含客户、账户或内部审批信息。

## 部署

项目为纯静态 Vite 站点，可部署到 GitHub Pages、Cloudflare Pages、Vercel 或 Netlify。

- 构建命令：`npm run build`
- 输出目录：`dist`
